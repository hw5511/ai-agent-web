Game Features

혼자서도 아름다운 여정

01

절차적 생성 우주

매 게임마다 달라지는 10억 개의 별계. 같은 경험은 두 번 없습니다. 알고리즘이 만드는 무한한 탐험지.

02

음악 반응형 환경

배경 음악의 비트와 멜로디에 반응하여 우주 환경이 실시간으로 변화합니다. 귀와 눈이 하나가 됩니다.

03

비선형 스토리텔링

정해진 순서가 없습니다. 탐험하는 순서에 따라 다른 이야기가 펼쳐집니다. 우주만큼 넓은 내러티브.
