수업 소개

AI 에이전트 웹 수업이란?

코딩 없이 AI 에이전트에게 지시만 해서 실제 브라우저에서 동작하는 웹 페이지를 만드는 수업입니다.
원하는 것을 말하면 AI 에이전트가 코드를 생성하고, 눈앞에서 완성된 결과물이 나옵니다.

코딩 지식 불필요

HTML, CSS, JavaScript를 몰라도 됩니다.
원하는 웹 페이지를 말로 설명하면 AI 에이전트가 코드를 생성합니다. 지시어(프롬프트)만으로 웹을 완성합니다.

코드가 눈에 보인다

AI 에이전트가 생성한 HTML/CSS/JS 코드를 직접 보게 됩니다. 코드가 어떻게 구조화되어 있는지, 어떻게 동작하는지 자연스럽게 이해하게 됩니다.

고퀄리티 인터랙션

마우스 인터랙션, 스크롤 애니메이션, 3D 오브젝트, 파티클 효과까지. 고급 웹 기술도 AI 에이전트에게 지시하면 구현됩니다.

수업 경험

AI로 시작하는 웹 개발 입문

완성된 웹 페이지를 직접 만들어보는 경험이, 웹 개발의 가능성을 새롭게 보게 만듭니다.

2

코드를 배우는 데 시간이 걸리고, 그동안 눈에 보이는 결과물이 없습니다

3

웹 개발의 세계는 넓고, 맨땅에서의 진입은 쉽지 않습니다

2
코드 구조가 눈에 들어온다

웹의 가능성을 체험한 뒤, 본격적인 학습의 출발점이 됩니다

LESSON 01

1회차 — 기초 인터랙션

마우스와 스크롤 이벤트를 활용한 인터랙티브 웹 페이지를 AI 에이전트 지시만으로 완성합니다.

수업 목표

"원하는 것을 말하면 완성된다"는 경험을 처음으로 합니다. 코딩 없이 AI 에이전트와 대화하는 것만으로 실제 브라우저에서 동작하는 인터랙티브 웹 페이지를 완성하며, AI가 생성한 HTML/CSS/JS 코드가 어떻게 구조화되어 있는지 눈으로 익힙니다.

배우는 내용
  • AI 에이전트에게 웹 기능을 지시하는 프롬프트 작성법
  • 마우스 커서 커스터마이징 및 Magnetic(자기장) 버튼 구현
  • CSS 3D 변환을 이용한 플립 카드 호버 인터랙션
  • 마우스 위치 기반 스포트라이트(빛 번짐) 효과
  • 생성된 코드에서 HTML 구조와 CSS 역할 파악하기
수업 결과물 예시
LESSON 02

2회차 — 고급 애니메이션

GSAP, Swiper, AOS 등 실무에서 널리 쓰이는 애니메이션 라이브러리를 AI 에이전트로 적용합니다.

수업 목표

코드 없이도 현업 수준의 애니메이션을 구현할 수 있다는 것을 체험합니다. 스크롤에 연동된 패널 전환, 텍스트 리빌, 드래그 슬라이더 등 실무 프로젝트에서 자주 등장하는 인터랙션을 AI 에이전트 지시만으로 완성합니다.

배우는 내용
  • GSAP ScrollTrigger를 활용한 수평 스크롤 패널 구성
  • 스크롤 진행도에 따라 텍스트 색이 바뀌는 컬러 리빌 효과
  • Swiper.js 드래그·스와이프 슬라이더 구현
  • AOS(Animate On Scroll) 카드 등장 애니메이션 적용
  • 외부 라이브러리를 CDN으로 불러오는 방식 이해
수업 결과물 예시
LESSON 03

3회차 — 라이브러리 & 3D

파티클, 3D, WebGL까지 — 시각적으로 강렬한 인터랙티브 웹을 AI 에이전트로 구현합니다.

수업 목표

tsParticles, Three.js, VANTA.js 등 고급 시각 효과 라이브러리를 AI 에이전트로 구현합니다. 3D와 WebGL 기술이 어떻게 웹에 적용되는지 직접 눈으로 확인하며, 웹이 얼마나 강렬한 시각 경험을 만들 수 있는지 체감하는 수업입니다.

배우는 내용
  • tsParticles로 마우스에 반응하는 인터랙티브 파티클 구현
  • Three.js를 활용한 3D 오브젝트 로딩, 회전, 조명 제어
  • VANTA.js WebGL 인터랙티브 배경 효과 적용
  • 순수 Canvas 2D API로 별자리 연결 파티클 구현
  • 라이브러리 옵션을 프롬프트로 조정하는 방법
수업 결과물 예시
LESSON 04

4회차 — Awwwards 수상작 카피

세계적인 웹 어워드 Awwwards SOTD 수상작을 스크린샷만으로 AI 에이전트가 재현합니다.

수업 목표

원본 코드 없이 이미지(Vision) 정보만으로 고퀄리티 결과물을 만드는 경험을 합니다. 수상작의 스크린샷을 AI 에이전트에게 보여주고 스타일을 재현하는 방식으로, AI 에이전트가 실무 수준의 디자인을 구현할 수 있다는 것을 직접 확인합니다.

배우는 내용
  • Vision 기반 프롬프트: 이미지를 주고 스타일을 재현하는 방법
  • 컬러·타이포그래피·레이아웃을 프롬프트로 지시하는 기술
  • 럭셔리 브랜드, 에이전시, 영화, 게임, 환경 등 다양한 스타일 구현
  • 결과물을 수정·개선하는 반복 프롬프팅 방법
  • AI 에이전트가 실무 수준의 결과물을 만드는 능력 직접 확인
