Introduction

AI 에이전트란?

스스로 판단하고,
행동하는 AI

AI 에이전트는 단순히 질문에 답하는 AI를 넘어, 스스로 목표를 설정하고, 계획을 세우고, 도구를 사용해 실행하는 자율적 AI 시스템입니다.

01
환경 인식

파일, 웹, API 등 외부 환경을 스스로 탐색하고 정보를 수집

02
자율 판단

수집한 정보를 바탕으로 다음 행동을 스스로 결정

03
도구 활용

코드 실행, 파일 편집, 외부 서비스 호출 등 실제 작업을 수행

04
반복 학습

결과를 확인하고 부족하면 스스로 수정하며 목표를 달성할 때까지 반복

단 한 번의 설정으로 완전 자동화

Curriculum

AI 에이전트 커리큘럼이란?

프로그래밍 경험이 없어도 실무에서 바로 활용 가능한 AI 에이전트를 직접 만들고 운영하는 과정입니다.

에이전트 기반 실습

설치형 AI 에이전트를 직접 설치하고 에이전트 환경에서 실습하며 익히는 과정

비개발자 대상

프로그래밍 경험이 없어도 AI 에이전트를 활용할 수 있도록 설계된 과정

3단계 로드맵

프롬프트 엔지니어링 → AI 에이전트 기초 → AI 에이전트 심화로 이어지는 체계적 구성

실무 자동화

메신저 연동, 업무 자동화, 웹 자동화 등 실무에서 바로 활용 가능한 내용

총 15회차 × 2시간  |  기초 8강 + 심화 8강으로 구성되며, 매 회차 실습 위주로 진행됩니다.
수강 완료 후 나만의 AI 에이전트 자동화 시스템을 직접 구축할 수 있습니다.

Basic

AI 에이전트 기초 커리큘럼

Claude Code 설치부터 Sub Agent, MCP, Skills 구축까지 — AI 에이전트를 다루는 핵심 기초 8강

1

AI 에이전트 이해와 CLI 환경 구축

2시간
  • AI 에이전트란? / 설치형 AI란?
  • Gemini CLI 설치 및 로그인
  • GPT CLI(Codex) 설치 및 로그인
  • Claude Code 설치 및 로그인
  • 세션 사용량 및 extra-usage
2

Claude 에이전트 메모리와 파일/폴더 제어

2시간
  • 에이전트 메모리 (/init, /memory, Auto Memory)
  • 에이전트 이름 지어주기 - 프롬프팅
  • 파일/폴더 생성, 이동, 이름변경
  • 파일/폴더 삭제, 복사, 목록 조회
  • 파일 내용 수정, 실행(start)
3

Claude Code 환경 설정과 기본 명령어

2시간
  • Claude 개인정보 보안 설정 체크하기
  • Claude 설정 세팅하기 (파일 삭제 금지 세팅)
  • Bypass 모드 / esc / 사전 입력
  • /clear, /login, /logout, /exit, /color
  • 웹 Search 툴 사용하기
  • Todos 기능 / /todos 명령어
4

세션 관리와 Node.js 백그라운드 작업

2시간
  • /resume, /context 명령어
  • /compact & auto compact
  • /rename 세션
  • /task - 백그라운드 프로세스 개념
  • 백그라운드 프로세스 만들기
5

Sub Agent와 Hooks

2시간
  • Sub Agent란?
  • Sub Agent의 작동 원리
  • Sub Agent 만들기
  • Claude in PowerPoint & Excel
6

MCP 개념 이해와 Notion 연동

2시간
  • MCP란?
  • MCP 설치 및 세팅 방법
  • Notion MCP 설치하기
  • Notion 에이전트 제어 실습
7

Skills 활용과 Discord 알림 연동 실습

2시간
  • Skills란?
  • Skill 사용하기
  • Discord 연동 Skill 제작 실습
  • 작업 완료 알림 받기
8

Claude Desktop 사용법

2시간
  • 일반 Claude Desktop
  • Desktop Claude Code
  • Claude Cowork
  • Claude Code CLI와 비교

Advanced

AI 에이전트 심화 커리큘럼

문서 자동화부터 웹 크롤링, 외부 API 연동, 워크플로우 자동화까지 — 실전 활용 8강

1

엑셀 & 워드 문서 자동 작성

2시간 — Tool & Automation
  • openpyxl로 엑셀 Skill 제작
  • python-docx로 워드 Skill 제작
  • 두 Skill을 파이프라인으로 연결
2

웹 브라우저 자동 조작하기

2시간 — Web Automation
  • 웹 자동화 원리 + Playwright 환경 구성
  • 페이지 이동 / 클릭 / 입력 자동화
  • 데이터 수집 및 CSV·JSON 저장
3

외부 API 연동 1: Google Cloud Platform

2시간 — Tool & Automation
  • OAuth 인증 공통 가이드 (OAuth 2.0 / 서비스 계정)
  • Sheets·Forms·Docs·Drive API 연동
  • Gmail API + Sheets 데이터 → Gmail 리포트
4

외부 API 연동 2: AI이미지 + Gemini + CLI

2시간 — Tool & Automation
  • AI 이미지 생성 Skill 제작
  • Gemini API 멀티모달 (텍스트·이미지 분석)
  • 이미지 생성 → 분석 → 저장 파이프라인
5

CLAUDE.md 트리거 + 문서 자동화

2시간 — Workflow Automation
  • CLAUDE.md 트리거 키워드 설정 + 문서 라우팅
  • 매뉴얼·가이드·PPT 자동 생성 실습
6

자동 트렌드 리서치 보고서 파이프라인

2시간 — Workflow Automation
  • RSS·웹 크롤링으로 트렌드 데이터 수집 + Claude 분석
  • Discord Webhook 자동 보고서 전송 파이프라인
7

네이버 블로그 자동 포스팅

2시간 — Workflow Automation
  • 주제 리서치 + 이미지 생성 API로 비주얼 소재 제작
  • 네이버 블로그 API 연동 + 자동 발행
8

Notion DB 기반 자동 에이전트

2시간 — Workflow Automation
  • Notion DB 폴링으로 변경사항 감지 + 자동 작업 실행
  • 결과 Notion 자동 보고 + Discord 알림

Use Cases

AI 에이전트 활용 사례

수강 후 직접 구현하게 될 실제 자동화 사례들입니다.

Email 01

이메일 자동 작성

컨텍스트 분석 후 Gmail 자동 작성 및 발송. 수신 메일에 맞게 답장을 생성하고 일정을 잡아줍니다.

Web 02

웹 브라우저 자동 조작

Playwright로 웹사이트 로그인, 폼 입력, 데이터 수집 등 반복 업무를 완전 자동화합니다.

Document 03

엑셀·워드 문서 자동화

데이터를 읽어 보고서를 자동 생성. 매주 자동으로 엑셀 매출 분석표 + 워드 요약본을 만듭니다.

Blog 04

네이버 블로그 자동 포스팅

트렌드 키워드 리서치부터 이미지 생성, 본문 작성, 자동 발행까지 원클릭으로 처리합니다.

Report 05

트렌드 리서치 보고서

RSS와 웹 크롤링으로 업계 트렌드를 수집, Claude가 분석해 Discord로 매일 자동 전송합니다.

Notion 06

Notion DB 자동 에이전트

Notion DB의 변경을 감지해 자동으로 작업을 실행하고, 결과를 Notion과 Discord로 보고합니다.
