AI 에이전트 웹 수업이란?

코딩 없이 AI 에이전트에게 지시만 해서 실제 브라우저에서 동작하는 웹 페이지를 만드는 수업입니다.
원하는 것을 말하면 AI 에이전트가 코드를 생성하고, 눈앞에서 완성된 결과물이 나옵니다.

코딩 지식 불필요

HTML, CSS, JavaScript를 몰라도 됩니다.
원하는 웹 페이지를 말로 설명하면 AI 에이전트가 코드를 생성합니다. 지시어(프롬프트)만으로 웹을 완성합니다.

고퀄리티 인터랙션

마우스 인터랙션, 스크롤 애니메이션, 3D 오브젝트, 파티클 효과까지. 고급 웹 기술도 AI 에이전트에게 지시하면 구현됩니다.

LESSON 01

1회차 — 웹의 언어를 만나다

HTML · CSS · JS가 무엇인지 이해하고, AI 에이전트 지시만으로 실제 랜딩페이지를 처음부터 완성합니다.

수업 목표

코드를 몰라도 "AI 에이전트가 대신 써준다"는 경험을 처음으로 합니다. HTML(뼈대)·CSS(스타일)·JS(행동)의 역할을 이해하고, 구조와 스타일을 분리해서 지시하는 방법을 익히며 랜딩페이지 한 장을 완성합니다.

배우는 내용
  • HTML · CSS · JavaScript 세 언어의 역할 분담
  • AI 에이전트로 랜딩페이지 뼈대 먼저 → CSS 입히기
  • 구조와 스타일을 분리해서 지시하는 방법
  • JS 버튼 클릭 인터랙션 추가
  • 지시 → 생성 → 확인 → 수정 사이클 체험

2회차 — 색상과 폰트로 퀄리티를 높인다

1회차 결과물에 색상 팔레트와 폰트를 입혀 Before/After 차이를 직접 체험합니다.

수업 목표

같은 구조라도 색상과 폰트 하나가 퀄리티를 완전히 바꾼다는 것을 체험합니다. 색상이 분위기를 만드는 원리를 이해하고, Google Fonts 조합을 AI 에이전트로 적용하며 1회차 결과물을 "성형수술"하는 Before/After 비교를 직접 만들어봅니다.

배우는 내용
  • 색상 팔레트 선정 원리 (주색·배경·강조색)
  • AI 에이전트로 색상 테마 일괄 변경
  • Google Fonts 조합 선택 및 적용
  • Before/After 비교로 디자인 차이 확인
  • 나만의 색상·폰트 조합 완성

LESSON 03

3회차 — 스크롤 트리거

GSAP ScrollTrigger로 스크롤에 연동되는 수평 패널 전환과 텍스트 컬러 reveal을 구현합니다.

수업 목표

스크롤이 단순한 위아래 이동이 아닌 애니메이션의 트리거가 될 수 있다는 것을 체험합니다. GSAP ScrollTrigger로 수평 스크롤 패널을 구성하고, 텍스트가 스크롤 진행도에 따라 색이 바뀌는 컬러 reveal 효과를 AI 에이전트로 만들어봅니다.

배우는 내용
  • ScrollTrigger 개념 — 스크롤이 애니메이션을 트리거하는 원리
  • GSAP ScrollTrigger 기본 사용법 (scrub, pin)
  • 수평 스크롤 패널 구현 (translateX 연동)
  • 스크롤 진행도에 따라 텍스트 색이 바뀌는 컬러 reveal
  • CDN으로 GSAP 라이브러리 불러오기

LESSON 04

4회차 — 마우스 인터랙션

마그네틱 버튼, CSS 3D Flip Card, Spotlight Follow — 마우스 이벤트 기반 인터랙션 3종을 AI 에이전트로 구현합니다.

수업 목표

마우스가 트리거가 되는 세 가지 인터랙션 패턴을 체험합니다. 자기장처럼 끌리는 마그네틱 버튼, 클릭하면 뒤집히는 Flip Card, 마우스를 따라 빛이 퍼지는 Spotlight 효과를 각각 지시만으로 완성합니다.

배우는 내용
  • mousemove 이벤트로 마그네틱(자기장) 버튼 구현
  • CSS 3D 변환을 이용한 Flip Card 호버 인터랙션
  • 마우스 위치 기반 Spotlight Follow 효과
  • 호버 · 클릭 · 드래그 트리거 구분과 활용
  • AI 에이전트로 인터랙션 옵션 조정하기

LESSON 05

5회차 — 라이브러리 + 3D

tsParticles, Typed.js, Three.js — 혼자서는 못 만들었을 효과를 AI 에이전트 지시 한 줄로 완성합니다.

수업 목표

전문 개발자도 시간이 걸리는 파티클·타이핑·3D 효과를 AI 에이전트로 즉시 구현합니다. 라이브러리가 무엇인지, CDN으로 어떻게 불러오는지 이해하고 "혼자 못 만들었을 것 같은" 결과물을 직접 완성하는 경험을 합니다.

배우는 내용
  • CDN 라이브러리 활용법 — script 태그 한 줄로 기능 추가
  • tsParticles로 마우스에 반응하는 인터랙티브 파티클 구현
  • Typed.js 타이핑 애니메이션 — 문구가 자동으로 타이핑되는 효과
  • Three.js 3D 오브젝트 — 회전하는 큐브/구 구현
  • 라이브러리 옵션을 프롬프트로 조정하는 방법
수업 결과물 예시
LESSON 06

6회차 — 배경 효과 + Canvas

VANTA.js WebGL 배경과 Canvas 2D 별자리 파티클 — 페이지 전체를 살아있는 배경으로 채웁니다.

수업 목표

웹 페이지의 배경 자체가 인터랙티브한 시각 경험이 될 수 있다는 것을 체험합니다. VANTA.js로 WebGL 기반 동적 배경을 적용하고, 순수 Canvas 2D API로 별자리 연결 파티클을 직접 구현합니다.

배우는 내용
  • VANTA.js WebGL 인터랙티브 배경 효과 적용
  • Canvas 2D API로 별자리 연결 파티클 구현
  • 배경 효과 선택 가이드 — 어떤 상황에 어떤 효과가 맞는가
  • 나만의 배경 조합 완성
  • 배경과 콘텐츠 레이어 분리 구조 이해

LESSON 07

7회차 — 기획 = 프롬프트

나쁜 프롬프트와 좋은 프롬프트의 결과물 차이를 직접 체험하며 퀄리티를 높이는 지시법을 익힙니다.

수업 목표

같은 AI 에이전트에게 지시해도 프롬프트 품질에 따라 결과물이 완전히 달라진다는 것을 체험합니다. 나쁜 프롬프트와 좋은 프롬프트를 직접 입력해보고 차이를 비교하며, 웹 기획이 곧 프롬프트 작성이라는 것을 이해합니다.

배우는 내용
  • 나쁜 프롬프트 분석 — 왜 결과가 나쁜가
  • 좋은 프롬프트 구조 — 목적 + 대상 + 스타일 + 기술스펙
  • 나쁜/좋은 프롬프트 직접 체험 비교
  • 퀄리티 차이 분석 — 무엇이 결과를 바꾸는가
  • 통합 랜딩페이지 실습
