AI 에이전트 웹 수업이란?

코딩 없이 AI 에이전트에게 지시만 해서 실제 작동하는 웹 페이지를 만드는 수업입니다.
원하는 것을 말하면 AI 에이전트가 코드를 생성하고, 즉시 배포 가능한 결과물이 나옵니다.

✨
코딩 지식 불필요

HTML, CSS, JavaScript를 몰라도 됩니다.
원하는 웹 페이지를 말로 설명하면 AI 에이전트가 코드를 생성합니다. 지시어(프롬프트)만으로 웹을 완성합니다.

🚀
즉시 배포 가능

GitHub Pages를 통해 만든 웹 페이지를 바로 인터넷에 공개할 수 있습니다. 수업 당일 완성한 결과물을 URL로 공유하는 경험을 합니다.

🎨
고퀄리티 인터랙션

마우스 인터랙션, 스크롤 애니메이션, 3D 오브젝트, 파티클 효과까지. 고급 웹 기술도 AI 에이전트에게 지시하면 구현됩니다.

AI vs 기존 방식

AI 에이전트, 웹 제작의 판을 바꾼다

기존 웹 제작 방식과 AI 에이전트 방식의 차이를 직접 확인하세요.

기존 방식
코드 직접 작성
1
코드 직접 작성

HTML/CSS/JS 문법을 공부하고 직접 코드를 작성해야 합니다

2
에러 수정 반복

오타 하나로 전체가 망가지고, 에러 원인을 찾는 데 시간이 소요됩니다

3
디자인 별도 학습

디자인 감각과 CSS 레이아웃 지식을 추가로 익혀야 합니다

4
학습 기간 수개월

기초부터 배워 실제 결과물을 내기까지 수개월이 필요합니다

높은 진입 장벽 — 코딩 경험 없이는 시작하기 어렵습니다
AI 에이전트 방식
원하는 것 설명만으로
1
원하는 것 설명

만들고 싶은 웹 페이지를 자연스러운 말로 설명합니다

2
AI 에이전트가 코드 생성

AI 에이전트가 HTML/CSS/JS 전체를 자동으로 작성합니다

3
즉시 결과 확인

브라우저에서 바로 확인하고, 마음에 안 들면 추가 지시로 수정합니다

4
당일 배포 완료

수업 당일 완성한 결과물을 GitHub Pages로 즉시 공개합니다

낮은 진입 장벽 — 지시하는 방법만 배우면 누구나 시작 가능
프롬프트 예시
"검정 배경에 마우스를 따라 빛이 퍼지는 인터랙티브 웹 페이지 만들어줘. 중앙에 큰 타이틀 텍스트가 있고 마우스를 올리면 글자가 흩어지는 효과도 넣어줘."
→ AI 에이전트가 HTML/CSS/JS 전체를 생성 — 즉시 브라우저에서 동작

LESSON 01

1회차 — 기초 인터랙션

마우스, 스크롤, 호버를 이용한 기초 인터랙티브 웹을 AI 에이전트로 제작합니다.

🖱
마우스 따라다니는 커서 효과

커스텀 커서와 자기장 버튼을 구현합니다. 마우스가 버튼에 가까워지면 버튼이 끌려오는 Magnetic 효과를 AI 에이전트가 순수 JS로 생성합니다.

📜
스크롤 연동 타이핑 텍스트

스크롤 위치에 따라 텍스트가 타이핑되는 애니메이션을 구현합니다. IntersectionObserver를 활용한 등장 효과도 함께 적용합니다.

🎯
CSS/JS 호버 카드 인터랙션

마우스를 올리면 카드가 기울어지거나 빛이 따라오는 고급 호버 효과를 구현합니다. CSS 변환과 JS 이벤트를 조합한 결과물입니다.

배포 URL https://hw5511.github.io/ai-agent-web/13_class_agent_v5/lesson1_ex1/
LESSON 02

2회차 — 고급 애니메이션

GSAP, Swiper, AOS 등 강력한 애니메이션 라이브러리를 AI 에이전트로 적용합니다.

📌
GSAP ScrollTrigger 스크롤 핀

세로 스크롤에 맞춰 가로로 이동하는 수평 스크롤 패널을 구현합니다. GSAP의 ScrollTrigger 플러그인으로 정밀한 스크롤 제어가 가능합니다.

🔁
Swiper.js 드래그 슬라이더

터치/드래그 인터랙션을 지원하는 프로 수준의 슬라이더를 AI 에이전트로 제작합니다. 자동 재생, 네비게이션, 페이지네이션이 포함됩니다.

⬆
AOS 스크롤 등장 애니메이션

스크롤을 내릴 때 각 섹션이 자연스럽게 등장하는 효과를 구현합니다. Animate On Scroll 라이브러리를 활용해 다양한 등장 방향을 설정합니다.

배포 URL https://hw5511.github.io/ai-agent-web/13_class_agent_v5/lesson2_ex1/
LESSON 03

3회차 — 라이브러리 & 3D

파티클, 3D, WebGL까지 — 시각적으로 강렬한 인터랙티브 웹을 AI 에이전트로 구현합니다.

✨
tsParticles 파티클 인터랙션

마우스를 따라 반응하는 파티클 효과를 구현합니다. tsParticles 라이브러리로 눈 내리는 효과, 버블, 연결선 파티클 등을 설정할 수 있습니다.

📦
Three.js 3D 오브젝트

브라우저에서 3D 그래픽을 렌더링하는 Three.js를 활용합니다. 회전하는 3D 구체, 와이어프레임 오브젝트 등을 AI 에이전트가 생성합니다.

🌊
VANTA.js WebGL 배경

WebGL 기반의 인터랙티브 배경 효과를 적용합니다. 파도, 새 떼, 안개 등 다양한 WebGL 배경을 한 줄의 지시로 적용할 수 있습니다.

🌟
Canvas 2D 파티클 (순수 JS)

별자리처럼 연결되는 파티클을 순수 JS Canvas로 구현합니다. 마우스 접근 시 반응하고, 커서 주변 파티클이 연결되는 인터랙션을 AI 에이전트가 작성합니다.

배포 URL https://hw5511.github.io/ai-agent-web/13_class_agent_v5/lesson3_ex1/
LESSON 04

4회차 — Awwwards 수상작 카피

Awwwards SOTD(Site of the Day) 수상작을 스크린샷만으로 AI 에이전트가 재현합니다.
원본 코드 없이 시각 정보만으로 고퀄리티 결과물을 만드는 경험입니다.

Result 01
AUPALE VODKA

어두운 갈색 고급 음료 브랜딩. 보틀 SVG 애니메이션과 골드 액센트 컬러, 럭셔리 타이포그래피가 결합된 프리미엄 주류 브랜드 사이트.

Result 02
DULCEDO

검정과 골드의 럭셔리 에이전시. 세련된 모션과 미니멀한 레이아웃으로 에이전시의 고급스러운 아이덴티티를 표현한 사이트.

Result 03
1820 PRODUCTIONS

파티클 효과가 인상적인 영화 프로덕션 사이트. 어두운 배경에 수많은 파티클이 움직이며 시네마틱한 분위기를 연출합니다.

Result 04
GRIFLAN

판타지/게임 브랜드 스타일. 중세 판타지 감성의 텍스처와 강렬한 타이포그래피, 드라마틱한 배경 효과가 특징입니다.

Result 05
OCEANX 2025

심해 탐험 인터랙티브 사이트. 깊은 바다 색감과 물속 파티클 효과로 탐험 다큐멘터리의 세계관을 웹으로 구현합니다.

Result 06
LIGHTWEIGHT

슈퍼 미니멀 자전거 휠 브랜드. 극도로 절제된 화이트 스페이스와 정밀한 제품 포커싱으로 제품의 가치를 극대화한 디자인.

결과물 비교 허브 https://hw5511.github.io/ai-agent-web/#awwwards
