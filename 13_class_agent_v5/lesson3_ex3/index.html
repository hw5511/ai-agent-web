About VANTA.js

VANTA.js란?

VANTA.js는 Three.js와 WebGL을 기반으로 동작하는 오픈소스 배경 애니메이션 라이브러리입니다. 새 떼, 파도, 안개, 별빛 등 다양한 자연 현상을 인터랙티브하게 웹 배경에 적용할 수 있습니다.

마우스 움직임에 반응하는 실시간 3D 렌더링으로 방문자에게 몰입감 있는 경험을 제공합니다.

다른 VANTA 효과들

Birds 외에도 다양한 효과를 지원합니다. 각 효과는 독립된 모듈로 로드됩니다.

VANTA.WAVES
Ocean / Fluid

부드럽게 물결치는 3D 파도 효과. 색상, 파고, 속도를 자유롭게 조절하여 깊고 몰입감 있는 바다 분위기를 연출합니다.

waveHeight • waveSpeed • shininess
VANTA.FOG
Atmosphere / Mist

서로 다른 색상의 안개가 천천히 흘러다니는 효과. 미스터리하고 몽환적인 분위기 연출에 최적화되어 있습니다.

highlightColor • midtoneColor • speed
VANTA.GLOBE
3D / Network

회전하는 3D 구체 위에 네트워크 연결선이 그려지는 효과. 테크, 글로벌, 데이터 관련 사이트에 어울리는 미래지향적 배경입니다.

color • backgroundColor • size
