ABOUT MODEL

Damaged Helmet은 Khronos Group이 공개한 대표적인 glTF 2.0 샘플 모델입니다.

PBR(Physically Based Rendering) 재질과 고해상도 텍스처를 사용하여 실제 금속 표면의 반사와 긁힘, 손상 질감을 WebGL 위에서 재현합니다.

Three.js의 GLTFLoader로 로드하며, 씬에 배치된 여러 광원이 금속 표면에서 반사되며 실시간으로 렌더링됩니다.

Format glTF 2.0 Binary
Textures Albedo · Normal · AO
Material PBR Metallic-Roughness
Renderer Three.js WebGLRenderer
Lighting Ambient · Directional · Point
