01 — How It Works

PURE JS
+ CSS VARS

마우스 위치를 mousemove 이벤트로 감지합니다. lerp(선형 보간)을 적용해 좌표값을 부드럽게 추종하고, requestAnimationFrame 루프로 매 프레임 CSS 변수를 갱신합니다.

오버레이 div의 radial-gradient가 CSS 변수를 읽어 실시간으로 빛의 위치를 이동시킵니다. GPU 합성 레이어만 사용하므로 60fps가 유지됩니다.

// mousemove → lerp → CSS var update
document.addEventListener('mousemove', e => {
  targetX = e.clientX;
  targetY = e.clientY;
});

// RAF loop
currentX += (targetX - currentX) * 0.08;
root.style.setProperty('--mouse-x', currentX + 'px');
root.style.setProperty('--mouse-y', currentY + 'px');
02 — Technology

ZERO
LIBRARY

01

MOUSEMOVE

브라우저 네이티브 이벤트만 사용. 외부 라이브러리 없이 순수 JS로 마우스 좌표를 실시간 추적합니다.

02

CSS CUSTOM
PROPERTY

--mouse-x, --mouse-y 변수를 JS에서 갱신하면 CSS가 자동으로 radial-gradient 위치를 반영합니다.

03

LERP

선형 보간(Linear Interpolation)으로 현재 좌표를 목표 좌표로 부드럽게 수렴시킵니다. factor 0.08 적용.

04

RAF LOOP

requestAnimationFrame으로 매 프레임 실행. 브라우저 리페인트 타이밍에 정확히 맞춰 60fps를 보장합니다.
